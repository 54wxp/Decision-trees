To compile the program, go into the root folder and make sure the java file is present.
1. javac viterbiAlg.java
 
To run the code.
1. java viterbiAlg

Once the training is done, the program becomes interactive. Please give it the input you want as per the options it displays.


Overview of the structure of the code.

1. Trains on the brown dataset.
2. 2 implementations - test on a test set, - test on your own sentence.
3. Implements an application when you press 1.


The root folder contains the Dataset link.txt that contains the link to download the data set.
